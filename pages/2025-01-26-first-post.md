# 블로그를 시작하며

안녕하세요! 새로운 블로그를 시작하게 되었습니다. 이 블로그는 GitHub Pages와 Vanilla JavaScript로 만들어진 정적 블로그입니다.

## 블로그 소개

이 블로그는 다음과 같은 기술로 구축되었습니다:

- **GitHub Pages**: 무료 호스팅
- **Markdown**: 게시글 작성
- **marked.js**: 마크다운 파싱
- **Prism.js**: 코드 하이라이팅
- **Giscus**: GitHub Discussions 기반 댓글 시스템

## 주요 기능

### 1. 다크모드 지원

우측 상단의 버튼을 클릭하여 다크모드와 라이트모드를 전환할 수 있습니다. 선택한 테마는 로컬 스토리지에 저장되어 다음 방문 시에도 유지됩니다.

### 2. 검색 및 필터링

메인 페이지에서 게시글을 검색하고 태그별로 필터링할 수 있습니다. 검색은 제목, 내용, 태그를 모두 포함합니다.

### 3. 댓글 시스템

각 게시글 하단에는 Giscus를 통한 댓글 시스템이 있습니다. GitHub 계정으로 로그인하여 댓글을 남길 수 있습니다.

## 앞으로의 계획

앞으로 이 블로그에서는 다음과 같은 주제들을 다룰 예정입니다:

- **웹 개발**: JavaScript, TypeScript, React 등
- **백엔드**: Node.js, API 설계
- **DevOps**: Docker, CI/CD
- **개발 팁**: 생산성 향상 도구 및 방법
